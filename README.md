# Sprint_6_Videogames_Sales

El objetivo del proyecto es: Encontrar patrones para detectar juegos exitosos que generan mayores ganancias a una empresa de venta de videojuegos, observando los géneros, plataformas y regiones.

En este proyecto realiza el análisis de los datos de venta de videojuegos de una compañia de ventas ficticia. Se realiza una limpieza de datos, visualizaciones utilizando distintas gráficas, agrupaciónes para análisis y pruebas de hipótesis.

El proyecto está realiado en Jupyter Notebook, utilizando lenguaje Python. Los datos utilizados se encuentran en el folder 'datasets'.
